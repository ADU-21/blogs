# CloudFormation简介
通过前面的学习我们已经知道了AWS为我们提供服务的方式是将现实世界中的各种需求抽象为很多服务，CloudFormation是一种被定义为管理工具的服务，其作用是把AWS提供的服务按一定方式组合起来，自动化而且可纳入版本管理，作为基础设施及代码的典型应用
